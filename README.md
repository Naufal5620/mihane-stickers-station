# Mihane Stickers Station (MSS)

## Overview

**Mihane Stickers Station (MSS)** is a sticker maker tool inspired by the Genshin Impact Sticker Maker, originally developed by [TheOriginalAyaka](https://github.com/TheOriginalAyaka/sekai-stickers). MSS extends the original concept by adding a customizable game filter, making it compatible with a broader range of anime RPG games. Now, users can create stickers for various popular titles, including:

- **Genshin Impact**
- **Honkai: Star Rail**
- **Zenless Zone Zero**
- **Wuthering Waves**
- **And more!**

## Features

- **Customizable Game Tags**: MSS allows you to filter stickers based on specific games, ensuring you get exactly what you're looking for.
- **Wide Anime RPG Support**: In addition to Genshin Impact, MSS supports multiple anime RPGs, offering a broader selection of stickers.
- **Easy-to-Use Interface**: Create your favorite stickers with a simple, user-friendly process.
- **Regular Updates**: The author will strive to keep all stickers up-to-date, ensuring new characters are added as they are released.
